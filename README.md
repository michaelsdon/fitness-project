## Fitness Project
The goal of this project is to provide a trainer-client based application to promote healthier lifestyles.

There are two sides to the project - Client side and Trainer side

**Trainer Side** - *features*

- Track clients and their progress
- Create unique content and plans per client
- Given a link for easy on boarding of current clients
- Automatically pushed onto the trainer search for potential clients
- Create custom diet and workout plans for each client

**Client Side** - *features*

- Workout schedule
- Contact trainer
- Select a trainer that works best in your needs.
